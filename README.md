Giới thiệu
Website tin tức thể thao trong nước là một hệ thống web được xây dựng nhằm cung cấp, cập nhật và quản lý các thông tin, bài viết và sự kiện thể thao diễn ra tại Việt Nam. Hệ thống hướng đến việc mang lại nguồn tin nhanh chóng, chính xác và dễ tiếp cận cho người dùng quan tâm đến các lĩnh vực thể thao trong nước.
-	Mục tiêu dự án
+ Xây dựng một website tin tức thể thao có cấu trúc rõ ràng, dễ sử dụng
+ Áp dụng kiến thức phân tích – thiết kế hệ thống vào thực tế
+ Thực hành thiết kế UML (Class Diagram), ERD, và Functional Diagram
+ Rèn luyện kỹ năng làm việc nhóm và phát triển ứng dụng web
-	Đối tượng sử dụng
+ Người dùng (User): Xem tin tức, tìm kiếm bài viết, bình luận, theo dõi lịch sử đọc
+ Quản trị viên (Admin): Quản lý nội dung, người dùng, danh mục, quảng cáo và xử lý báo cáo
-	Các chức năng chính
+ Xem và tìm kiếm tin tức thể thao theo danh mục
      + Đăng ký, đăng nhập và quản lý tài khoản người dùng
+ Bình luận và tương tác với bài viết
+ Quản lý bài viết, hình ảnh, video
+ Phân quyền người dùng và quản trị hệ thống
+ Gửi thông báo, thống kê lượt xem
+ Quản lý quảng cáo và xử lý báo cáo nội dung
Dự án bao gồm các sơ đồ phân tích và thiết kế:
+ Sơ đồ lớp (OOP / UML Class Diagram): Mô tả các lớp, thuộc tính, phương thức và mối quan hệ
+ Sơ đồ thực thể – quan hệ (ERD): Mô hình hóa cơ sở dữ liệu với các thực thể, khóa chính và khóa ngoại
+ Sơ đồ chức năng (Functional Diagram): Biểu diễn các chức năng chính và chức năng con của hệ thống
-	Công nghệ sử dụng ( dự kiến )
+ Frontend: HTML, CSS, JavaScript
+ Backend: PHP
+ Database: MySQL
+ Công cụ thiết kế sơ đồ: draw.io
+ Quản lý mã nguồn: GitHub
-	Trạng thái dự án

Dự án đang trong quá trình phân tích và thiết kế hệ thống, phục vụ cho đồ án cuối kỳ môn Phát triển ứng dụng Web.
